# 知识库 RAG 助理 - 示例文档

这是一个示例文档，用于演示知识库问答系统的功能。

## 产品简介

知识库 RAG 助理是一个基于 LangChain 和 OpenAI 构建的智能问答系统。它能够：

1. **文档管理**：支持上传和管理 Markdown 和纯文本文档
2. **智能检索**：使用向量数据库进行语义搜索
3. **精准问答**：基于 RAG（检索增强生成）架构提供准确回答
4. **来源追溯**：每个回答都附带参考来源

## 技术架构

### 核心组件

- **FastAPI**：高性能的 Python Web 框架
- **LangChain**：大语言模型应用开发框架
- **ChromaDB**：本地向量数据库
- **OpenAI API**：提供语言模型和 Embedding 服务

### 工作流程

1. 用户上传文档到知识库
2. 系统将文档切分为小块
3. 使用 OpenAI Embeddings 将文本块向量化
4. 向量存储到 ChromaDB
5. 用户提问时，系统检索相关文档块
6. 将检索结果和问题一起发送给 LLM
7. LLM 生成基于知识库的回答

## 使用场景

- 企业内部知识库问答
- 技术文档助手
- 客户服务智能回复
- 学术研究资料查询

## 注意事项

1. 确保上传的文档内容准确、权威
2. 定期更新知识库以保持信息时效性
3. 复杂问题可能需要多轮对话来澄清

## 常见问题

**Q: 支持哪些文件格式？**
A: 目前支持 `.md`（Markdown）和 `.txt`（纯文本）格式。

**Q: 如何提高回答质量？**
A: 上传更多相关文档，确保文档内容详实、结构清晰。

**Q: 可以同时询问多个问题吗？**
A: 建议一次询问一个问题，这样可以获得更精准的回答。
